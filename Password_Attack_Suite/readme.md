## 📖 Overview

Passwords are the first line of defense in any system. However, weak passwords make systems vulnerable to attacks.  
This project simulates real-world password attack techniques such as:

- Dictionary Attack  
- Brute Force Attack  
- Hash-Based Password Cracking  

It also evaluates password strength using entropy and generates a security report.

---

## 🎯 Features

- 🔹 Custom Password Dictionary Generator  
- 🔹 Password Strength Analyzer (Entropy-based)  
- 🔹 Brute Force Attack Simulation  
- 🔹 Hash Generation & Cracking (MD5)  
- 🔹 Security Report Generation  
- 🔹 CLI-based Interactive Interface
