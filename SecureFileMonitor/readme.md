# 🔐 Secure File Monitoring System

A Python-based cybersecurity project that monitors sensitive files in real-time and detects unauthorized access or modifications using SHA-256 hashing.

---

## 📌 Project Overview

The **Secure File Monitoring System** is designed to protect sensitive files by continuously monitoring a selected directory. It detects file activities such as creation, modification, deletion, and movement.

The system focuses on maintaining **data integrity** and identifying **unauthorized changes** using cryptographic hashing techniques.

---

## 🎯 Objectives

- Monitor files in real-time  
- Detect file creation, modification, deletion, and movement  
- Ensure file integrity using SHA-256 hashing  
- Identify unauthorized access or modifications  
- Generate alerts and maintain logs  

---

## ⚙️ Technologies Used

- Python  
- Watchdog Library  
- Hashlib (SHA-256)  
- OS Module  
- Time Module  

---

## 🚀 Features

- 🔄 Real-time file monitoring  
- 🔐 SHA-256 based integrity checking  
- ⚠️ Unauthorized modification detection  
- 📢 Alert generation system  
- 📝 Event logging with timestamps  
- 🚫 Debounce mechanism to avoid duplicate alerts
