# 🔐 Threat Intelligence Aggregator

## 📌 Project Overview
The Threat Intelligence Aggregator is a cybersecurity project designed to collect, analyze, and evaluate potential Indicators of Compromise (IOCs) such as URLs and IP addresses. The system helps in identifying whether a given input is safe, suspicious, or malicious based on predefined intelligence sources.

This project demonstrates how threat intelligence can be automated and used to enhance security analysis and decision-making.

---

## 🎯 Objectives
- To analyze URLs and IP addresses for potential threats  
- To detect suspicious or malicious indicators  
- To generate a structured security report  
- To simulate real-world threat intelligence workflows  

---

## ⚙️ Features
- ✅ URL Analysis  
- ✅ IP Address Detection  
- ✅ Blacklist Checking  
- ✅ Simple Threat Classification (Safe / Suspicious / Malicious)  
- ✅ Automated Report Generation  
- ✅ Easy-to-use Command Line Interface  

---

## 🛠️ Technologies Used
- Python 3  
- Regular Expressions (re module)  
- OS & File Handling  
- Basic Cybersecurity Concepts
